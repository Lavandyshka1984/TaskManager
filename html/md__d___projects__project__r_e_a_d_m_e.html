Task Manager
Постановка задач и описание проекта

Это групповой проект о системе управления задачами. У нас есть 2 базы данных: 1)Cотрудники. 2)Задачи. Для реализации поставленной задачи у нас есть 4 класса:

  • 1-Employee
  • 2-Task
  • 3-Database (это также будет шаблон)
  • 4-Manager
  • Предполагается, что в программе будут функции:.
  • 1)Для класса Database.
  • Функция загрузки бд в файл и из него.
  • Функция взятия размера массива.
  • Функция удаления элемента.
  • 2)Для класса Manager.
  • Функция обновления баз данных.
  • Функция присвоения таски к работнику.
  • Просмотра базы данных сотрудников.
  • Просмотра базы данных тасков.
  • Добавление сотрудника.
  • Удаление сотрудника.
  • Изменение данных сотрудника.
  • Добавление Task'а.
  • Удаление Task'а.
  • Изменение данных Task'а.
  • Поиск сотрудника по id или имени , или по задаче , по выбору.
  • Поиск Task'а по id или названию , по выбору.
  • Прошло n дней (пользуется штуками от класса Manager).
